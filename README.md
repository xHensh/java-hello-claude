# java-hello-claude

A simple Hello World Java program generated by [Claude](https://claude.ai), Anthropic's AI assistant.

## Requirements

- Java 25+

## Run

```bash
javac HelloClaude.java
java -Dstdout.encoding=UTF-8 HelloClaude
```

## Output

```
Hello, World! 👋
This program was written by Claude, Anthropic's AI assistant.
Running on Java 25
```
